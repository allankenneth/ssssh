About

This website is not affiliated with the City of Victoria, British Columbia in any way and is not guaranteed to be accurate. The display is based on a reading of the bylaws and a custom data representation derived from that. You can view the code on GitHub .

This was created in the spirit that more people would follow the bylaws and be respectful of their neighbors if they had an easy way to check the bylaws that didn't involve reading a 20 page PDF to find the relevant section relative to the current date and time.
